# RPG em Java 🎮

Projeto simples de RPG por turnos desenvolvido em Java.

## Funcionalidades
- Escolha de classe (Guerreiro, Mago, Arqueiro)
- Sistema de combate por turnos
- Inimigos aleatórios
- Interface com JOptionPane

## Tecnologias
- Java
- POO (Programação Orientada a Objetos)
